= DataCalendar

== Description

Generates the data needed to represent a given schedule a calendar. 

== Installation

    sudo gem install data_calendar

== Usage

    require 'data_calendar'
    
    dc = DataCalendar.new(:date => Time.now -1.year)
    
    pp dc.all_days #=>
